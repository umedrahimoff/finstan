# Finstan — учёт финансов

## Стек

- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Zustand — состояние
- React Hook Form + Zod — формы
- TanStack Table — таблицы
- Recharts — графики
- react-big-calendar — календарь
- date-fns — даты

## Запуск

```bash
npm install
npm run dev
```
